🐛 TROUBLESHOOTING

SE RECEBER ERRO: "Cannot find module"
→ Verifique o caminho do import
→ Certifique-se que o arquivo está no lugar correto

SE RECEBER ERRO: "Property does not exist"
→ Adicione useEstoqueManager() no componente
→ Passe todos os props para EstoqueRefiner

SE RECEBER ERRO: "Toast not found"
→ Se usar toast, importe corretamente
→ Ou mude alert() se necessário

SE A TABELA NÃO CARREGAR:
→ Abra DevTools (F12) → Console
→ Verifique se há erro de conexão Supabase
→ Verifique se suas chaves estão corretas
